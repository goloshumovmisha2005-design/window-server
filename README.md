## Компетенции

### Active Directory и управление пользователями
- Установка и настройка основного и резервного контроллеров домена (Windows Server 2022).
- Создание организационных подразделений (OU), групп безопасности, пользователей (вручную и через PowerShell).
- Настройка доверительных отношений между доменами (One-way Trust, Two-way Trust).
- Делегирование прав администраторам подразделений.
- Управление SID, ввод клиентских машин в домен.

### Сетевые службы (DNS, DHCP)
- **DNS**: настройка зон прямого и обратного просмотра, условная пересылка (Conditional Forwarders), CNAME-записи.
- **DHCP**: создание пулов адресов, резервирование (Reservations), управление через PowerShell (Get-DhcpServer, Set-DhcpServer).
- Настройка резервного DHCP-сервера для отказоустойчивости.

### Групповые политики и безопасность (GPO)
- Парольные политики (сложность, срок действия, блокировка учётных записей).
- Централизованное управление брандмауэром (Windows Defender Firewall) через GPO.
- Управление антивирусом Windows Defender через GPO.
- Централизованное журналирование (Windows Event Collector, WinRM).
- Разграничение прав доступа к файловым ресурсам (Access-Based Enumeration).
- Автоматическая установка ПО через GPO (Software Installation).

### PKI и VPN
- Развёртывание инфраструктуры открытых ключей (корневой и подчинённый центр сертификации).
- Выпуск и установка сертификатов для аутентификации.
- Настройка защищённых VPN-соединений (IKEv2/IPsec с аутентификацией по сертификатам).
- Настройка PPTP-соединений.
- Настройка безопасного доступа к FTP (FTP over SSL, SFTP через OpenSSH-Win).

### Файловые сервисы и отказоустойчивость
- Создание RAID-массивов (RAID 1 и RAID 5) через Diskpart и Disk Manager.
- Настройка распределённой файловой системы (DFS) для отказоустойчивости.
- Управление квотами (Hard Quota, Soft Quota).
- Теневые копии (VSS) для восстановления файлов.
- Резервное копирование (Windows Server Backup).
- Настройка SMB-протокола (отключение устаревших версий SMBv1).

## Удалённый доступ (IIS, FTP, RemoteApp)
- Установка и настройка IIS (Internet Information Services).
- FTP-сервер (обычный и с SSL/TLS).
- RemoteApp и удалённый рабочий стол (RDP / Remote Desktop Services).
- Публикация приложений через RemoteApp.
- Настройка теневого подключения (Shadow RDP).


### Инструменты автоматизации
- **PowerShell** — скрипты для создания пользователей, настройки DHCP, управления GPO и бэкапов.
- **Server Core** — работа с Windows Server в режиме без графического интерфейса.
